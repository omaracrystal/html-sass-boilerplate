#HTML SASS Boilerpate

**Sass folders structure**
```
- global
	    - base
	    - helpers
	    - mixins
	    - variables
- site
	    - components
		        - base-components
		        - component-name.scss
	    - pages
		        - partials
		        - page-name.scss
		
```

###To get started

```
git clone https://github.com/omaracrystal/html-sass-boilerplate.git
```
```
npm install
```
```
npm start
```
